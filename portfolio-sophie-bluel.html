Site de Sophie Bluel, architecte d'intérieur

Ce site est le portfolio en ligne de l'architecte d'intérieur Sophie Bluel. Sa réalisation a été pour moi un vrai rite de passage puisqu'il m'a initié au langage Javascript, incontournable pour tout développeur "frontend" (c'est à dire qui travaille sur le rendu visible du site). Ce projet comporte une galerie d'images que l'on peut filtrer par catégorie, une interface connecté/déconnecté fonctionnant via un formulaire de connexion, et même une boîte modale pour ajouter ou supprimer des images (disponible en mode connecté uniquement). Ce projet m'a également initié au traitement de données récupérées depuis une API, ce qui est une compétence fondamentale pour tout développeur Web qui se respecte !

Technologies utilisées : HTML5, CSS3, JAVASCRIPT, FIGMA, API, NODE.JS.
